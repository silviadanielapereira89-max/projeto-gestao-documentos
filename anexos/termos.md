# Glossário de Termos Técnicos

## A
**Ata** - Documento que regista decisões e ações de uma reunião

## C
**Commit** - Registo de alterações no controlo de versões  
**Clone** - Cópia local de um repositório remoto

## P
**Pull Request** - Pedido de integração de alterações  
**Push** - Enviar alterações locais para repositório remoto

## R
**Repositório** - Local de armazenamento de código e documentação  
**Requisito** - Necessidade ou funcionalidade que o sistema deve cumprir
