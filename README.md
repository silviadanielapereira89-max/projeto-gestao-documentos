# Projeto de Gestão de Documentos

Este repositório centraliza toda a documentação de um projeto de desenvolvimento, incluindo planeamento, atas de reuniões, relatórios e comunicação interna. O objetivo é manter um histórico completo e organizado de todas as atividades e decisões.

## Estrutura de Pastas

projeto-gestao-documentos/ ├── planeamento/ # Objetivos, cronogramas e planos de execução ├── atas/ # Atas de todas as reuniões ├── relatorios/ # Relatórios semanais e finais ├── comunicacao/ # Contactos e mensagens da equipa └── anexos/ # Requisitos, glossário e documentos de suporte


## Como Contribuir

1. **Nunca fazer commits diretos na branch main**
2. Criar sempre uma nova branch para cada dia/tarefa
3. Fazer Pull Request para integração após validação
